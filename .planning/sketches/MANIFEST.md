# Sketch Manifest

## Design Direction
AI-chat-first portfolio for chrisonline.nl. A cinematic brand intro **morphs into** an AI-chat
hero: the existing hexagon mark scales/blurs and *becomes* an iridescent orb, which sits as a
soft **halo behind** the greeting + glassmorphism prompt field + generic capability chips.
On scroll the page leads with **Werk** (proof first), then the **"over mij" + cover photo**
block mid-page, then **Diensten**. Dark base (#0a0b15) with one bright accent (cyan #00B0D5),
ambient aurora glow, all over the existing mouse-reactive particle field.

## Reference Points
- Dribbble "LIX" AI-assistant home (orb + greeting + prompt + example chips) — the user's reference video.
- Existing site: IntroOverlay cinematic opening, ParticleField canvas, Inter, brand cyan.

## Orb decision
**Hexagon-becomes-orb morph (002B)** + **orb-as-halo in the hero (001B)**. Orb engine to be
finalized in build (Paper Shaders mesh-gradient is the lead; CSS orb is the lightweight
fallback). The CDN-loaded Paper Shaders mesh orb was validated in-sketch.
