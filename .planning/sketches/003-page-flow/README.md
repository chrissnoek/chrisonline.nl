# Sketch 003: Pagina-flow & 'over mij'

## Design Question
The chat is the new hero — so what comes *below* it on scroll, and crucially where do the
**"over mij" block + cover photo** land so they're not lost?

## How to View
open .planning/sketches/003-page-flow/index.html — scroll each variant.

## Variants
- **A: Over mij ná hero** — hero → **About + cover** → Werk → Diensten. About gets prime position right after the chat.
- **B: Werk eerst, over mij mid** — hero → Werk → **About + cover** → Diensten. Leads with proof, personality mid-page.
- **C: Chat dockt** — same order as A, but once you scroll past the hero the **chat shrinks into a fixed corner dock** so it's always reachable while browsing.

## What to Look For
- Does the cover photo + "Beschikbaar"-badge feel prominent enough (it's a key part of the current site's identity)?
- Variant C dock: helpful "always-on assistant", or distracting?
- The note under Werk: real projects render as inline cards when the agent calls `showProjects`.
- Note: "Shopify webshop-development" appears under Diensten — the agent can pitch Shopify as a *service* even though there's no Shopify *project* yet (validates generic chips).
